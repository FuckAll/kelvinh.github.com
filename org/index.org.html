* Description

  This is the personal site of Kelvin Hu, and it is also a demo site for
  org-page, a static site generator for Emacs and org mode, written by
  =Elisp=. There are many popular static site generators, such as jekyll,
  octopress, etc. Most of them support =markdown=, but seems none of them
  supports written in org mode, so, org-page is born for those who use both
  Emacs and org mode, and also want to publish their achievements to a site
  which holds static pages, such as github pages.

* 说明

  这是Kelvin Hu的个人网站，同时也是org-page的一个示例站点。org-page是一个用 =Elisp= 写
  的，为Emacs和org mode准备的静态站点生成器。现在有很多流行的静态站点生成器，像jekyll，
  octopress等等，它们大多数支持使用 =markdown= 来写作，但是似乎没有能支持org mode写作
  的，于是，org-page就这样诞生了，如果你是Emacs和org mode的使用者，并且想把你写的作品发
  布到一个支持静态页面的站点，例如github pages，那么，你就可能是org-page的潜在用户。
